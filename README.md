# 🎬 ScreenMatch

Projeto desenvolvido em **Java** com foco no treinamento e consolidação de conceitos fundamentais da linguagem e de orientação a objetos.

---

## 📚 Objetivo

Praticar os principais fundamentos do Java por meio da construção de uma aplicação de busca e exibição de informações sobre filmes, reforçando lógica de programação e organização em pacotes.

---

## 🛠️ Tecnologias Utilizadas

- Java 24
- IntelliJ IDEA
  
---

## 📈 Status do Projeto

🚧 Em desenvolvimento — projeto voltado para aprendizado e evolução técnica em Java Back-End.
